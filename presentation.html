¿Qué es React?

  • ⚛️ Librería de JS para construir UI
  • 📦 Desarrollada por Meta (Facebook)
  • 🔁 Usa componentes para construir la UI de forma modular
  • 🔄 flujo de datos unidireccional
  • 🧠 Forma declarativa

Ventajas de React

  • ✅ Basado en componentes reutilizables
  • ⚡ Actualizaciones rápidas con Virtual DOM
  • 🎯 Unidireccional: flujo de datos predecible
  • 🔧 Gran ecosistema y comunidad
  • 🧠 Ideal para aplicaciones interactivas y complejas
  • 📦 Integración sencilla con otras bibliotecas o frameworks
  • 📱 Compatible con desarrollo móvil (React Native)

¿Qué es JSX?

  • 📝 Sintaxis que combina JavaScript y HTML
  • ⚙️ Transpilado a llamadas `React.createElement`
  • 🔒 Permite usar expresiones dentro del marcado
  • 📦 Soporta atributos, componentes y estructuras condicionales

JSX => JS


// JSX
const element = (
  <h1 className="saludo">Hola, mundo</h1>
);
  

// Equivalente sin JSX
const element = React.createElement(
  'h1',
  { className: 'saludo' },
  'Hola, mundo'
);

Escribir marcado con JSX

  • Sintaxis JSX
  • Expresiones dentro de JSX

const name = 'Ada';
const element = <h1>Hola, {name}</h1>;

Agregar estilos

  • Usar el atributo `style`

Mostrar datos

  • Interpolación

Listas

Eventos

  • Eventos en JSX
  • Funciones manejadoras

function handleClick() {
  alert('Hiciste clic');
}
<button onClick={handleClick}>Clic</button>

Actualizar la pantalla

  • useState

const [count, setCount] = useState(0);
<button onClick={() => setCount(count + 1)}>{count}</button>

Usar hooks

  • useState
  • useEffect

useEffect(() => {
  document.title = `Contador: ${count}`;
}, [count]);
